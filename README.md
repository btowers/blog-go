# Go Blog Backend

### **Description**

A Go backend for a blog app using JSON Web Token for authentication and mongoDB as database

### **Requirements**

- Go 1.15+
- MongoDB Community Server or MongoDB Atlas account (https://www.mongodb.com/)

### **Install**

```console
$ npm install
```
